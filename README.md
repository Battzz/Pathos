# Conductors

Brush version 1 of a local-first desktop app built with Tauri, React, Vite, and
Vitest.

## Scripts

- `npm run tauri dev` starts the desktop app in development mode.
- `npm run build` builds the frontend bundle.
- `npm run test` runs the current regression test suite.
- `npm run test:watch` starts Vitest in watch mode.

## Current Scope

This first brush intentionally keeps the app simple:

- one Tauri window
- one minimal React application shell
- one baseline test to lock the startup UI
